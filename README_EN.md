# Campus-Activity-Seeds V2.0 (校园活动信息港 V2.0)
> **This README_EN.md is Translated By Machine.**

## Project Description
This is a campus-oriented activity management and credit tracking platform, primarily designed to address pain points in the "second classroom" scenario. 
It aggregates various campus activities and allows activity organizers to publish events with one click by automatically parsing public account articles or group messages via AI. 
It helps students clearly keep track of their participation history and credit accumulation progress, aiming to avoid the awkward situation of being held back by insufficient credits right before graduation.

## Tech Stack (summarized by AI)
**Frontend**: Vue 3.5 + TypeScript 7 + Vite 8 + Element Plus + Tailwind CSS v4  
**Backend**: FastAPI (Python) + SQLAlchemy 2.0 + MariaDB  
**Authentication**: joserfc (Ed25519) + bcrypt  
**AI Service**: OpenAI SDK (I used DeepSeek)

## Feature Introduction
- This platform has four roles: Student, Activity Organizer, Administrator, and Super Administrator.
- Students can search for activities, filter them, mark participation, save favorites, etc.
- Activity organizers can publish/edit activities, use AI to assist in filling out forms, upload event photos, etc.
- The Administrator has quite a few features; you can check the screenshots below or the demo video.

### Starting Up (commands generated by AI)
**Backend Startup**:
1. Enter the backend directory
   `cd backend`
2. Create a virtual environment (first time)
   `python -m venv venv1`
3. Activate the virtual environment
Windows:`venv1\Scripts\activate`
macOS/Linux:`source venv1/bin/activate`
4. Install dependencies within the virtual environment
   `pip install -r requirements.txt`
5. Copy and edit environment variables (first time) (edit .env to fill in database password, JWT secret, AI API key, etc.)
   `cp .env.example .env`
6. Run database migrations
   `alembic upgrade head`
7. Start
   `uvicorn app.main:app --reload --host 127.0.0.1 --port 8000`

**Frontend Startup**
cd frontend  ->  npm install  ->  npm run dev
