# Campus-Activity-Seeds

**本项目的灵感来源于凑了一个学校的比赛热闹，诶比赛的时候没有弄好然后成功得了一个“优秀奖”。后来觉得还是得把这个坑填了，于是就花了10多天的空余时间吧，把这个弄完了，还是能用的，就是比较简单**


作为校园生活的见证者，你可能会遇到这些场景：班级群里的活动通知总是石沉大海，想找活动信息却怎么也翻不到；科创竞赛、志愿活动、学术讲座四散各处，无人整合；当毕业季来临，自己的第二课堂学分却还是零散在记忆里（其实就是没达标）。 Campus Activity Seeds （校园信息活动港）正是为此而来。它聚合了校园内的科创、志愿、讲座与分享会，并通过 AI 自动解析公众号文章，帮助活动发布者一键生成活动信息。对于参与者来说，它更像一位温柔的记录者，帮你留住在每场活动中的足迹，把散落的每一次参与，都呈现为清晰的学分雷达图，让你看见自己在专业之外正在悄然生长的领域。 每一次伸手探索、每一次投身其中，都是在大学这片土壤里播下的一粒种子，终将长成独属于你的花期。因为每一步都算数。
