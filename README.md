# Campus-Activity-Seeds V2.0
## 项目说明：
这是一个面向高校的活动管理与学分追踪平台（主要是设计成第二课堂了）。这个可聚合校内各类活动，活动发布方可以通过 AI 自动解析公众号文章/群消息一键发布活动，帮助在校学生清晰记录参与足迹和学分累积进度，尽量避免出现人马上毕业却被分值卡住的尴尬情景。

## 背景介绍：
这个的出现是因为上一个月在学校蹭了一个ai黑客松编程比赛，然后想出的这个想法，然鹅因为比赛答辩前没折腾好所以崩了，没做出啥功能。现在的这个是比赛结束后又抽了10多天空闲时间并跟ai斗智斗勇折腾出来的。其实我一点也不懂编程，也不是计算机专业的。这个项目融合了一些评委老师们的建议，但是什么自动爬取公众号我是真的不会啊。（其实这个项目还是有很多瑕疵/bug的,但是也没啥空了，就放在这做一个纪念这样。）

## 技术栈（AI定的）：
**前端**：Vue 3.5 + TypeScript 7 + Vite 8 + Element Plus + Tailwind CSS v4  
**后端**：FastAPI (Python) + SQLAlchemy 2.0 + MariaDB  
**认证**：joserfc (Ed25519) + bcrypt  
**AI服务**：OpenAI SDK (我用的DeepSeek)

## 功能介绍:
- 此平台有四个角色，分别是学生、活动发布方、管理员、超级管理员
- 学生能够找活动、筛选活动、标记参与活动、收藏活动等
- 活动发布方可以发布/编辑活动，用ai辅助填写表单，塞活动照片等
- 管理员的功能有点多，诶可以看下边的截图或者发的演示视频

## 项目环境要求
- Python 3.14+、Node.js 24+、MariaDB 11.8+

### 启动（AI生成的命令）
**后端启动**：
1. 进入后端目录          
```
cd backend
```
2. 创建虚拟环境（首次）
```
python -m venv venv1
```
3. 激活虚拟环境
Windows:
```
venv1\Scripts\activate
```
macOS/Linux:
```
source venv1/bin/activate
```
4. 在虚拟环境下安装依赖
```
pip install -r requirements.txt
```
5. 复制并编辑环境变量（首次）(编辑 .env 填写数据库密码、JWT 密钥、AI API Key等）
```
cp .env.example .env
```
6. 执行数据库迁移
```
alembic upgrade head
```
7. 启动
```
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```
**前端启动**
cd frontend  ->  npm install  ->  npm run build  ->  npm run dev
